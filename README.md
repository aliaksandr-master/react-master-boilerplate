# react-master-boilerplate


## start
run: `npm run start`

run: `npm run start-sandbox`


## generator:

run `npm run gen`

run `npm run action`


## run all tests: 

run `npm run lint`

run `npm run jest`

run `npm t`


## update

run `npm run cleanup`


## build

run `npm run publish-dev`

run `npm run publish-dev-min`

run `npm run publish-prod`
